# Senti 测试用例

测试日期：2026-04-25
复测日期：2026-04-26（Metrics API）
测试环境：本地 Docker Compose，Web `http://localhost`，API `http://localhost/api`
自动化脚本：`scripts/e2e-smoke.sh`，运行说明见 `docs/testing.md`

## 测试范围

- 账号鉴权：注册、登录、退出、登录态持久化、未登录访问保护。
- 文本分析：默认样例、有效聊天文本、空文本、异常输入提示、结果展示。
- 截图分析：未选择图片提示、图片选择状态、图片上传分析、OCR 失败提示。
- 分析结果：阶段、摘要、指标、态度、心理状态、证据、建议、回复参考、风险提醒、结构化聊天记录。
- 历史记录：默认空态、保存分析、历史列表展示、打开详情、删除历史。
- API：健康检查、AI 可用性、鉴权接口、分析接口、保存/历史接口、错误码。
- 可观测性：Prometheus 指标入口、基础安全响应头。
- 前端可用性：主要按钮禁用状态、错误提示、页面刷新后的登录态。
