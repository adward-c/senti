# 核心算法加权处理模块

本模块将恋爱博弈中的感性体验拆解为多维度的底层参数，通过 10 大领域的跨学科交叉验证，构建出逻辑自洽的决策公式库

## 一、全局参数字典与跨学科底层映射

> 本表定义了系统运行的基础核心变量（所有参数取值范围通常在 `0.0 - 1.0` 之间，视具体情境加权）。在分析聊天记录时，系统需首先对这些变量进行量化赋值。

| 参数符号 | 理论含义 (Theoretical) | 现实含义 (Practical) | 所属领域 (Domain) | 参考著作/文献 (References) |
| :--- | :--- | :--- | :--- | :--- |
| `Sv` | 生存资源价值 (Survival) | 物质基础、社会地位、提供稳定性的能力 | 进化心理学 | 《自私的基因》、繁衍价值理论 |
| `Rv` | 繁衍情绪价值 (Reproductive) | 外貌吸引力、性张力、提供情绪刺激的能力 | 进化心理学 | 《自私的基因》、性选择理论 |
| `Pface` | 前台面子阻力 (Face) | 对方为维持社交形象而产生的防备、矜持与借口 | 微观社会学 | 《日常生活的自我呈现》、拟剧论 |
| `Backstage` | 后台暴露度 (Backstage) | 卸下防备后展示的不完美、私密、脆弱的一面 | 微观社会学 | 《日常生活的自我呈现》、拟剧论 |
| `Exp` | 心理预期 (Expectation) | 历史互动建立的习惯阈值（如习惯了被秒回） | 认知心理学 | 认知失调理论、预期管理 |
| `Act` | 实际体验 (Actual) | 当下的真实互动体验（如突然遭遇冷淡） | 认知心理学 | 认知失调理论、多巴胺奖赏回路 |
| `S_cost` | 沉没成本 (Sunk Cost) | 对方在你身上投入的时间、精力和情绪内耗 | 行为经济学 | 《思考，快与慢》、前景理论 |
| `L_aversion`| 损失厌恶系数 (Loss Aversion)| 人类面对失去时的痛苦指数（通常加权为 2.2 倍） | 行为经济学 | 《思考，快与慢》、前景理论 |
| `Anx` | 焦虑依恋值 (Anxiety) | 对方在亲密关系中对不确定性的恐惧程度 | 精神分析学 | 依恋理论 (Attachment Theory) |
| `Def` | 心理防御墙 (Defense) | 面对高吸引力目标时触发的自我保护与疏离机制 | 精神分析学 | 心理防御机制 (Defense Mechanisms) |
| `Cp` | 服从阶梯 (Compliance) | 对方对你提出的微小指令的顺从程度与次数 | 说服与影响力学 | 《影响力》、承诺与一致性偏误 |
| `Sp` | 显示性偏好 (Revealed Pref.) | 对方付出的实际代价（如推掉社交、熬夜陪聊） | 行为经济学 | 显示性偏好理论 |
| `Fback` | 反馈增益 (Feedback) | 系统的能量反馈量（回复字数、频率、主动性） | 控制论/系统论 | 稳态控制理论 |
| `Noise` | 言语掩饰 (Noise) | 嘴上说的反话、敷衍或推脱词汇 | 控制论/系统论 | 信噪比理论 (Signal-to-Noise Ratio) |
| `Ve` | 情绪效价 (Valence) | 词汇中蕴含的情绪浓度（从极度冷淡到极度激越） | 认知科学 | 情感计算原理 |
| `Internal_D`| 内在欲望 (Desire) | 对方被你吸引产生的探索欲和靠近欲 | 马克思辩证法 | 《矛盾论》、内因 |
| `External_R`| 外部阻力 (Resistance) | 世俗观念、环境压力、人设包袱等外部障碍 | 马克思辩证法 | 《矛盾论》、外因 |
| `Ddepth` | 战略纵深 (Depth) | 你的神秘感，保留的底牌，以及随时离场的权力 | 博弈/军事学 | 纳什均衡、战略纵深 |
| `Rlatency` | 响应延迟 (Latency) | 对方回信速度与你回信速度的动态比值 | 权力博弈论 | 重复博弈中的权力动态 |
| `P_succ` | 成功概率 (Probability) | 发起升温邀约时的预估成功率 | 概率与风险学 | 期望值理论 (Expected Value) |
| `P_fail` | 破裂风险 (Risk) | 动作失败导致关系倒退或崩盘的概率 | 概率与风险学 | 期望值理论、黑天鹅效应 |

## 二、十大领域底层公式库

### 领域 1：进化心理学 (基础吸引力)
**公式 1：综合吸引力基数 (Base_Attraction)**
> **逻辑：** 无论男女，初始吸引力均由“生存护航能力”和“情绪繁衍能力”加权构成。
> **算法：** `Base_Attraction = (Sv * 0.4) + (Rv * 0.6)`

### 领域 2：微观社会学 (面子与害羞)
**公式 2：真实亲密度 (IS - Intimacy Score)**
> **逻辑：** 针对“端着”的测试。她越愿意暴露不完美的后台，关系越真。
> **算法：** `IS = Backstage / (Pface + 1.0)`

### 领域 3：认知心理学 (情绪落差感)
**公式 3：情绪落差刺激 (Gap_Effect)**
> **逻辑：** 解释“欲擒故纵”。制造负向预期差，引发对方主动消除认知失调的冲动。
> **算法：** `Gap_Effect = Act(当前体验) - Exp(历史预期)`

### 领域 4：行为经济学 (吃亏与沉没成本)
**公式 4：沉没成本锁定指数 (SCL - Sunk Cost Lock-in)**
> **逻辑：** 离不开低价值目标，是因为“吃亏”触发了损失厌恶。
> **算法：** `SCL = (投入时间 + 情绪内耗 + 妥协次数) * L_aversion`

### 领域 5：两性精神分析学 (安全感防御)
**公式 5：心理防御破拆率 (DW - Defense Wall Breakdown)**
> **逻辑：** 面对回避型的高防御对象，高吸引力反而会触发疏远，需用安全感降解防御。
> **算法：** `DW_Breakdown = IS / (Anx * Def)`

### 领域 6：说服与影响力学 (惯性服从)
**公式 6：服从阶梯指数 (Cp_Index)**
> **逻辑：** 必须从小成本服从开始，积累“一致性偏误”才能顺利完成大邀约。
> **算法：** `Cp_Index = 成功服从的小指令数 / 总指令数`

### 领域 8：马克思唯物辩证法 (破局临界点)
**公式 7：矛盾演化状态 (CS - Contradiction State)**
> **逻辑：** 激化矛盾打破舒适区。量变引起质变，找准临界点升温。
> **算法：** `CS = Internal_D(内在欲望) - External_R(外部阻力)`

### 领域 10：概率与风险学 (期望值评估)
**公式 8：升温期望值 (EEV - Escalation Expected Value)**
> **逻辑：** 发起动作前，计算收益与下行风险的综合期望值。
> **算法：** `EEV = (P_succ * 升温情绪红利) - (P_fail * 势能降级风险)`

## 三、核心实战决策逻辑流 (Core Decision Engines)

> 本模块为 AI 执行管线的“大脑”，整合跨学科推演，附带严格的**触发阈值**。

### 1. 意图真实度判定 (IVI - Intent Veracity Index)
> **理论依据：** 控制论 (过滤噪声) + 行为经济学 (显示性偏好)
> **用途：** 穿透“反话”和“假性拒绝”，过滤面子客套话。
> **逻辑算法：**
> `IVI = [Sp * log(Fback + 1)] / [User_Investment * Pface]`
> 
> **执行与阈值判定：**
> 只要行为投入(`Sp`)足够大，即使嘴上拒绝，IVI 也会拉高。
> * **`IVI > 1.0`**：口嫌体正直。判定为**真实好感**，可忽略口头否定，强势带领。
> * **`IVI < 0.5`**：行为与言语双重冷淡。判定为**真实没戏**，立即止损。

### 2. 社交势能监测 (SPE - Social Potential Energy)
> **理论依据：** 博弈论 (重复博弈) + 军事学 (战略纵深)
> **用途：** 监控权力平衡，防止沦为低位“舔狗”。
> **逻辑算法：**
> `SPE = (User_Ddepth / Target_Ddepth) * (Target_Latency / User_Latency)`
> 
> **执行与阈值判定：**
> 谁回信更慢、保留的神秘信息更多，谁的势能就越高。
> * **`0.8 < SPE < 1.5`**：健康均势区间，正常博弈拉扯。
> * **`SPE < 0.6`**：高危低位区间。**【红线阻断】**AI 强制要求停止输出情感话术，启动“断联/极简回复”机制。

### 3. 升温窗口期计算 (EWS - Escalation Window Score)
> **理论依据：** 认知科学 (情绪差) + 说服学 (服从性)
> **用途：** 决定何时发起邀约，降低被拒风险。
> **逻辑算法：**
> `EWS = (Gap_Effect * Cp_Index) + EEV - 稀缺性损耗`
> 
> **执行与阈值判定：**
> 窗口期是动态积累的，久聊无情绪起伏会因“稀缺性损耗”归零。
> * **`EWS > 0.8`**：出击信号。立刻输出“模糊邀约”或“线下肢体升温”指令。

## 四、专家组高级增量推论 (Advanced Deductions)

> 基于动态系统演化出的补充测算，用于复杂和极端局势的判定。

### 1. 情绪波动捕捉 (EV - Emotional Volatility)
* **逻辑算法：** `EV = StDev(Ve)` （计算历史情绪效价的标准差）
* **原理判定：** 对方产生的情绪波动越大（哪怕是吵架、生气、吐槽），说明你的心理权重越高。如果互动总是保持平淡的礼貌（`EV 接近 0`），则是绝对的社交死局。

### 2. 终极行动总线 (Action Protocol)
* **逻辑算法：** `Action = Function(IVI, SPE, EWS, EV, CS)`
* **策略分发规则：**
  1. **【进攻指令】** (EWS 高，IVI > 0.8)：直接发难，给出邀约/测试话术。
  2. **【拉扯指令】** (SPE 平衡，IVI 及格，但 EWS 未满)：利用 `Gap_Effect`，给出调侃、推拉话术，积攒 `EV`。
  3. **【重置指令】** (SPE < 0.6 或 IVI < 0.5)：执行降维打击。给出体面收尾、提供台阶或直接撤退建议，重建 `Ddepth`。
